##################################
#このパッケージについて
##################################
windows で動くpython環境をインストール不要なようにまとめたもの。
python, pip, vscode（エディタ）がインストール不要で使える。



##################################
#前準備
##################################

１：解凍したフォルダ（shinjin_kensyu_2019）の中の　
　　VSCode-win32-x64-1.33.1 \ Code.exe  を起動
２：左側サイドバー内の”虫眼鏡の上のアイコン”をクリック
３：「Open folder」をクリックし、”解凍したフォルダ”を選択して、「選択」をクリック
４：コマンドプロンプトを開いて、解凍したフォルダまでディレクトリを移動する
５：以下のコマンドを打って実行する
setup.bat
６：以下のコマンドを打って実行する
python main.py
７：カメラ映像が出てきたら成功！ESCキーを押して、アプリを停止。前準備終了。



##################################
#pip で XXX をインストールする方法

python -m pip install XXX
